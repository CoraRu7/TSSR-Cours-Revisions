# **Les opérateurs et opérateurs logiques**

## **Objectif**

Comprendre et manipuler les opérateurs arithmétiques, de comparaison et logiques afin de créer des programmes conditionnels fiables.

---

## **Concepts clés**

### **I) Opérateurs arithmétiques et de comparaison**

#### 1. Opérateurs arithmétiques

* **Addition** : `+`
* **Soustraction** : `-`
* **Multiplication** : `*`
* **Division** : `/`
* **Modulo** : `%` (reste de la division entière)
* **Exponentiation** : `**` (puissance)

**Exemples :**

```pseudo
2 + 3  // 5
15 % 2 // 1
3 ** 4 // 81
```

#### 2. Priorité des opérateurs

1. `**` (exponentiation)
2. `* / %` (multiplication, division, modulo)
3. `+ -` (addition, soustraction)

* Respecter l’ordre gauche → droite pour opérateurs de même priorité.
* Les parenthèses `( )` permettent de forcer l’ordre des calculs.

**Exemple :**

```pseudo
1 - 2 ** 3 / 4 + 3
// 2 ** 3 = 8
// 8 / 4 = 2
// 1 - 2 + 3 = 2
```

#### 3. Opérateurs de comparaison

* `==` : égal à
* `!=` : différent de
* `<` : inférieur à
* `<=` : inférieur ou égal à
* `>` : supérieur à
* `>=` : supérieur ou égal à

Renvoient **True** ou **False**.

#### 4. Exemples d’utilisation

**Exemple 1 : contrôle d’un achat**

```pseudo
m <- Saisir montant
s <- Saisir solde
Si s >= m Alors
    Afficher "Autoriser la transaction"
Sinon
    Afficher "Refuser la transaction"
FinSi
```

**Exemple 2 : détermination d’une mention**

```pseudo
n <- Saisir note
Si n >= 16 Alors Afficher "très bien"
Sinon Si n >= 14 Alors Afficher "bien"
Sinon Si n >= 12 Alors Afficher "assez bien"
Sinon Si n >= 10 Alors Afficher "passable"
Sinon Afficher "non obtenu"
FinSi
```

### **II) Opérateurs logiques**

#### 1. Algèbre de Boole

* Valeurs booléennes : `True` (Vrai), `False` (Faux)
* Principaux opérateurs :

| Opérateur   | Description                               | Symboles      |   |         |
| ----------- | ----------------------------------------- | ------------- | - | ------- |
| NON         | inverse la valeur                         | `!`           |   |         |
| ET          | vrai si toutes les conditions sont vraies | `&&` ou `and` |   |         |
| OU          | vrai si au moins une condition est vraie  | `             |   | `ou`or` |
| OU exclusif | vrai si une seule condition est vraie     | `xor`         |   |         |

#### 2. Exemples simples

```pseudo
Afficher (Faux ou Vrai)           // Vrai
Afficher ((Vrai ou Faux) et Vrai) // Vrai
Afficher ((Vrai xor Faux) ou !Vrai) // Vrai
```

#### 3. Exemple pratique

**Décision d’un étudiant pour sortir**

```pseudo
a <- Ai-je assez d'argent ? (o/n)
f <- Ai-je fini mes devoirs ? (o/n)
v <- Voiture dispo ? (o/n)
g <- Transports en grève ? (o/n)

Si a ET f ET (v OU NON g) Alors
    Afficher "Les conditions sont réunies pour sortir."
Sinon
    Afficher "Les conditions ne permettent pas de sortir."
FinSi
```

#### 4. Exemple avec boucle `while`

* Déterminer le point d’intersection de deux droites :

```php
$double = 30;
$triple = 10;
$max = 60;
while ($double > $triple && $double <= $max) {
    $double += 2;
    $triple += 3;
}
```

* Teste si intersection avant une valeur maximale.

## **Commandes / Notes importantes**

* Toujours utiliser les parenthèses pour clarifier l’ordre des opérations.
* Combiner les opérateurs logiques pour gérer plusieurs conditions.
* Les valeurs booléennes servent à prendre des décisions conditionnelles.
