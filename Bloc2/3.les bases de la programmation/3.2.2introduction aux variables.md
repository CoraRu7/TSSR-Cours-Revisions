# **Introduction aux variables**

## **Objectif**

Comprendre le rôle des variables en programmation et savoir les déclarer, manipuler et utiliser avec des opérateurs de base pour résoudre des calculs et gérer des données.

---

## **Concepts clés**

### **1. Contexte**

* **Pseudo-code** : langage abstrait pour représenter un algorithme sans se soucier de la syntaxe d’un langage de programmation spécifique.
* Permet de comprendre les mécanismes de programmation et de communiquer des idées clairement.

---

### **2. Définition d’une variable**

* Symbole ou nom représentant une **valeur inconnue ou changeante**.
* Sert à **stocker, manipuler et réutiliser des données** dans un programme.
* Chaque variable a :

  * Un **nom unique** (identifiant)
  * Un **type** (entier, flottant, chaîne, booléen…)
* Exemple mathématique :

  * `b = a + 3` → b dépend de a
  * `x + 2 = 3` → x = 1

---

### **3. Résolution d’équations et inéquations**

* **Équation simple** : isoler la variable

  * Ex. : `2x + 6 = 4x - 6` → `x = 6`
* **Inéquation** : même principe, mais attention au signe lors de la multiplication/division par un négatif

  * Ex. : `-2x + 4 > 8` → `x < -2`

---

### **4. Variables et fonctions**

* Une **fonction** associe un élément d’un domaine à un élément d’un codomaine.
* Notation : `f(x)` où `x` est la variable.
* Exemple : `f(x) = 2x + 1` → pour x=3, `f(3)=7`
* **Domaine** : ensemble des valeurs possibles pour la variable

  * Ex. : `f(x) = 1/x` → x ≠ 0

---

### **5. Déclaration et nommage**

* **Déclaration** : informer le compilateur ou l’interpréteur qu’une variable existe et réserver de la mémoire.
* **Nommage** :

  * Nom descriptif, court mais explicite
  * Respecter la convention du langage (camelCase, snake_case…)
  * Éviter les conflits avec mots-clés ou fonctions intégrées

#### **Exemples**

* Python : `taille = 5`
* C++ : `int age;`
* Java : `String nom;`
* JavaScript : `let prix = 10;`

---

### **6. Opérateurs arithmétiques**

| Opérateur | Fonction       | Exemple            |
| --------- | -------------- | ------------------ |
| +         | Addition       | `somme ← a + b`    |
| -         | Soustraction   | `diff ← a - b`     |
| *         | Multiplication | `produit ← a * b`  |
| /         | Division       | `quotient ← a / b` |
| %         | Modulo         | `reste ← a % b`    |

* **Attention** : division par zéro interdite.

---

### **7. Règles importantes**

* Déclarer une variable avant son utilisation.
* Le type de la variable influence les opérations possibles.
* Une variable peut être modifiée au cours de l’exécution du programme.
* Respecter la cohérence dans le nommage pour faciliter la lecture et la maintenance.

---

### **8. Essentiel**

* Les variables sont **fondamentales** en programmation : elles permettent de stocker et manipuler des données.
* Elles facilitent les calculs, la résolution d’équations et la représentation de fonctions.
* Les types principaux : **entiers, flottants, chaînes de caractères, booléens**.
* Le pseudo-code est un outil pratique pour comprendre les variables avant de coder dans un langage spécifique.
