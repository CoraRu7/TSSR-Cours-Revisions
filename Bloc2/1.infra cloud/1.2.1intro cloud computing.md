# Fiche Cloud Computing – Synthèse

## 1. Définition

Le **cloud computing** permet d’accéder à des services informatiques (logiciels, stockage, serveurs, etc.) **en ligne** via internet, au lieu de les installer localement.

* Avantages : réduction des coûts, flexibilité, accessibilité, moins de matériel nécessaire.
* Basé sur la **virtualisation** : plusieurs machines virtuelles peuvent coexister sur un même serveur.

---

## 2. Types de services

| Service                                | Contenu / Usage                                  | Exemple              |
| -------------------------------------- | ------------------------------------------------ | -------------------- |
| **IaaS** (Infrastructure as a Service) | Serveurs, stockage, réseau, machines virtuelles  | AWS EC2              |
| **PaaS** (Platform as a Service)       | Environnement de développement, outils logiciels | Google App Engine    |
| **SaaS** (Software as a Service)       | Applications prêtes à l’usage via navigateur     | Office 365           |
| **XaaS** (Everything as a Service)     | Tout autre service disponible en ligne           | Voir liste sur Auvik |

---

## 3. Types de déploiement

| Déploiement | Description                                           | Exemple                                                             |
| ----------- | ----------------------------------------------------- | ------------------------------------------------------------------- |
| **Public**  | Serveurs partagés, accessible via internet            | AWS, Microsoft Azure                                                |
| **Privé**   | Infrastructure dédiée, sécurisé et personnalisable    | Serveurs internes ou hébergés privés                                |
| **Hybride** | Combinaison public + privé, flexible et sécurisé      | Entreprise avec données sensibles locales et services cloud publics |
| **Autres**  | Poly Cloud, Multicloud, HPC, distribué, communautaire | Dépend du besoin spécifique                                         |

---

## 4. Vocabulaire clé

* **API** : interface permettant la communication entre applications.
* **ASP / FAH** : applications hébergées accessibles à distance.
* **Big Data** : très grandes données nécessitant des analyses spécialisées.
* **Data Center** : centre d’hébergement sécurisé de serveurs.
* **Data Scientist** : expert en analyse de données.
* **VPN** : réseau privé sécurisé sur internet.
* **Web Service** : protocoles permettant aux applications de communiquer.

---

## 5. Exemple : Amazon Web Services (AWS)

* Fournisseur majeur de cloud.
* Offre des services gratuits limités (EC2, S3, RDS, ELB).
* Attention aux dépassements : carte de crédit nécessaire, penser à retirer le moyen de paiement pour éviter les prélèvements automatiques.
