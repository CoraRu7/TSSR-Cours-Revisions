# Fiche – Introduction à Microsoft Azure

## 1. Contexte

* Microsoft Azure est une **plateforme cloud** proposant plusieurs services :

  * Stockage, machines virtuelles, intelligence artificielle, bases de données, etc.
* Accessible via un navigateur avec un **compte Microsoft**.
* Coût : dépend de l’usage, du temps et des services sélectionnés.

---

## 2. Création d’un compte Azure gratuit

1. Aller sur la page Microsoft Azure → **« Démarrer gratuitement »**.
2. Se connecter ou créer un compte Microsoft.
3. Remplir les informations (numéro de téléphone, carte bancaire).
4. Compte gratuit : **12 mois** ou crédit valable 30 jours.
5. Accéder au **portail Azure**.

---

## 3. Interface Azure

* **Volet gauche / barre de recherche** pour naviguer.
* Options principales en haut à gauche :

  1. **Azure Cloud Shell** – création de scripts.
  2. **Répertoire et abonnements**.
  3. **Notifications**.
  4. **Paramètres** – couleur, langue, etc.
  5. **Support et dépannage**.
  6. **Envoyer des commentaires** à Microsoft.
* **Tous les services** : accès à tous les services Azure, triés par catégorie.

---

## 4. Création d’une ressource (exemple VM Windows 10)

1. Cliquer sur **« Créer une ressource »**.
2. Rechercher la ressource → sélectionner **Microsoft Windows 10**.
3. Choisir la version désirée → cliquer sur **Créer**.
4. Configurer la VM :

   * Nom, disques, réseau, paramètres divers.
5. Cliquer sur **Vérifier + Créer** → VM prête à l’utilisation.

**Attention** : la création de VM est payante.

---

## 5. Points clés

* Azure permet de créer et gérer facilement des ressources cloud.
* Interface intuitive avec barre de recherche et accès à tous les services.
* Idéal pour tester des services cloud avec le compte gratuit.
* La VM est configurable et évolutive selon les besoins de l’utilisateur.
