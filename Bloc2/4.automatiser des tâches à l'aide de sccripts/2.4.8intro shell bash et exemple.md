# **Introduction au Shell Bash**

## **Objectif**

Comprendre le fonctionnement de Bash, savoir exécuter un script shell et connaître les principales commandes et concepts.

---

## **Concepts clés**

### **1. Définition de Bash**

* **Bash** : Bourne Again Shell, un shell Unix et langage de scripting.
* Permet d’exécuter des commandes en ligne de commande et d’automatiser ces commandes via des scripts.
* Disponible sur Linux, macOS (alternatif depuis 2019) et Windows.

### **2. Qu’est-ce qu’un shell ?**

* Interface texte permettant de contrôler l’ordinateur via des commandes, sans interface graphique.
* Avantages : rapide, automatisable, permet d’interagir avec toutes les ressources, traçable pour audit.

### **3. Utilité de Bash**

* Automatiser des tâches :

  * Renommer ou déplacer des fichiers,
  * Planifier des tâches,
  * Détecter des fichiers en double,
  * Implémenter une base de données via fichiers texte.
* Utilisable à distance via **SSH** (Secure Shell).
* Permet de lancer des processus en parallèle grâce aux pipes (`|`).

### **4. Concepts complémentaires**

* **Script** : ensemble de commandes exécutées par le système ou l’application.
* **Librairie** : collection de routines réutilisables.
* **PATH** : répertoires contenant des exécutables accessibles depuis n’importe où.

---

## **II) Exécution d’un script Bash**

### **1. Vérifier l’installation de Bash**

```bash
ps
```

* Permet de voir les processus en cours et vérifier si Bash est actif.

### **2. Lancer un script**

* Se placer dans le dossier contenant le script :

```bash
cd /chemin/du/script
```

* Lancer le script :

```bash
bash script.sh
# ou
./script.sh
```

* Rendre le script exécutable si nécessaire :

```bash
chmod +x script.sh
```

### **3. Comparaison des commandes Bash et PowerShell**

| Action                          | Bash     | PowerShell                |
| ------------------------------- | -------- | ------------------------- |
| Comparer deux valeurs           | `=` `!=` | `-eq` `-ne` `-ceq` `-cne` |
| Chercher un mot dans un fichier | `grep`   | `select-string`           |
| Substitution de chaîne          | `sed`    | `-replace`                |
| Afficher une valeur             | `echo`   | `Write-Host`              |
| Créer une variable              | `var=3`  | `$var = 3`                |

---

## **III) Essentiel**

* Bash est à la fois **shell et langage de script**.
* Permet **d’exécuter et d’automatiser** des commandes.
* Utile pour la gestion de fichiers, l’automatisation des tâches et le travail distant.
* Connaître la syntaxe et les commandes de base permet de créer des scripts efficaces et polyvalents.
