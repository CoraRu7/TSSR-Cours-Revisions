# **Prise en main du Shell Bash**

## **Objectif**

Apprendre les bases du Shell Bash et connaître les 10 commandes essentielles pour naviguer et manipuler les fichiers et processus.

---

## **Concepts clés**

### **1. Définition**

* **Bash** : Bourne Again Shell, un **shell Unix** et un **langage de scripting**.
* Permet d’exécuter des commandes en ligne de commande et d’automatiser des tâches via des scripts.
* Répertoire des commandes principales : `/usr/bin`, `/bin`.
* Structure d’une commande :

```
commande [arguments] [options]
```

* **Commande** : action à exécuter.
* **Arguments** : précisent l’objet de la commande (fichier, répertoire…).
* **Options** : modifient le comportement de la commande (ex : `-a`, `-r`).

---

## **II) 10 Commandes essentielles**

| Commande | Description                               | Options importantes / Exemple                               |
| -------- | ----------------------------------------- | ----------------------------------------------------------- |
| `su`     | Se connecter en **root**                  | `su` → demande le mot de passe                              |
| `cd`     | Changer de répertoire                     | `cd ..` (parent), `cd -` (précédent), `cd` ou `cd ~` (home) |
| `ls`     | Lister les fichiers du répertoire courant | `ls -a` (fichiers cachés), `ls -l` (liste longue)           |
| `touch`  | Créer un fichier vide                     | `touch fichier.txt`                                         |
| `rm`     | Supprimer un fichier                      | `rm -i` (confirmation), `rm -r` (récursif)                  |
| `mkdir`  | Créer un répertoire                       | `mkdir dossierTest`                                         |
| `more`   | Lire un fichier page par page             | `more fichier.txt`                                          |
| `grep`   | Rechercher une chaîne de caractères       | `grep "mot" fichier.txt`                                    |
| `ps`     | Afficher les processus en cours           | `ps u`, `ps ax`                                             |
| `kill`   | Terminer un processus                     | `kill PID` (PID obtenu via `ps`)                            |
| `tar`    | Créer ou exploiter une archive            | `tar -xvf archive.tar`                                      |

---

## **III) Essentiel**

* Bash permet de gérer **fichiers, répertoires et processus** via des commandes en ligne.
* Les commandes de base à connaître pour une utilisation quotidienne :

  * `su`, `cd`, `ls`, `touch`, `rm`, `mkdir`, `more`, `grep`, `ps`, `kill`, `tar`.
* Chaque commande peut avoir des **arguments** et des **options** pour adapter son comportement.
* Les scripts Bash automatisent ces commandes pour des tâches répétitives ou complexes.
