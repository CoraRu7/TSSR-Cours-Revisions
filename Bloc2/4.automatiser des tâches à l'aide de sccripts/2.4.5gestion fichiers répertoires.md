# **PowerShell : Gestion des fichiers et des répertoires**

## **Objectif**

Savoir créer, supprimer, déplacer et modifier des dossiers et fichiers à l’aide de PowerShell. Appliquer de bonnes pratiques pour organiser et sécuriser les données.

---

## **Concepts clés**

### **1. Gestion des dossiers**

* **Créer un dossier** :

```powershell
mkdir C:\Users\Eugenie\Documents\dossierTest
```

* **Afficher les fichiers dans un dossier** :

```powershell
Get-ChildItem C:\Users\Eugenie\Documents\dossierTest2
```

* **Supprimer un dossier** :

```powershell
Remove-Item C:\Users\Eugenie\Documents\dossierTest
```

**Conseils pratiques :**

* Toujours lancer PowerShell en **mode administrateur** pour éviter les restrictions.
* Il est possible de créer plusieurs dossiers simultanément avec `mkdir`.
* Utiliser une nomenclature cohérente pour organiser vos dossiers.

---

### **2. Gestion des fichiers**

* **Créer un fichier** :

```powershell
New-Item -Path C:\Users\Eugenie\Documents\dossierTest\test.txt
```

* **Écrire dans un fichier** :

```powershell
Add-Content C:\Users\Eugenie\Documents\dossierTest\test.txt 'Hello World'
```

* **Afficher le contenu d’un fichier** :

```powershell
Get-Content C:\Users\Eugenie\Documents\dossierTest\test.txt
```

* **Déplacer un fichier** :

```powershell
Move C:\Users\Eugenie\Documents\dossierTest\test.txt C:\Users\Eugenie\Documents
```

* **Supprimer un fichier** :

```powershell
Remove-Item C:\Users\Eugenie\Documents\test.txt
```

---

### **3. Bonnes pratiques**

* Créer un **modèle de dossiers et fichiers** pour normaliser la structure.
* Mettre en place une **convention de nomenclature** (noms clairs et cohérents).
* Utiliser un **outil de versionning** pour suivre les modifications.
* **Sauvegarder régulièrement** les systèmes et bases de données pour éviter toute perte.

---

### **4. Essentiel à retenir**

1. `mkdir` → créer un dossier
2. `Get-ChildItem` → lister les fichiers d’un dossier
3. `Remove-Item` → supprimer un dossier ou un fichier
4. `New-Item` → créer un fichier
5. `Add-Content` → ajouter du contenu dans un fichier
6. `Get-Content` → afficher le contenu d’un fichier
7. `Move` → déplacer un fichier
