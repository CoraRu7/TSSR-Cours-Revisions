# **Installation du Shell Bash**

## **Objectif**

Savoir vérifier si Bash est installé et connaître la méthode pour l’installer sur Windows ou Linux.

---

## **Concepts clés**

### **1. Définition de Bash**

* **Bash** : Bourne Again Shell, shell Unix et langage de scripting.
* Permet d’exécuter des commandes dans une interface texte et de créer des scripts automatisés.
* Présent par défaut sur la majorité des distributions Linux et macOS (alternatif depuis 2019).
* Disponible sous Windows depuis 2016.

---

## **II) Vérification de l’installation**

### **1. Sous Linux**

* Bash est **préinstallé** sur la plupart des distributions (ex: Ubuntu).
* Vérifier que Bash est actif :

```bash
ps
```

* Vérifier la version de Bash :

```bash
echo $BASH_VERSION
```

### **2. Sous Windows**

* Taper **`bash`** dans la barre de recherche du menu Démarrer.
* Si Bash apparaît, il est installé et peut être lancé.

---

## **III) Installation de Bash sous Windows 10**

### **1. Pré-requis**

* Vérifier que Windows 10 est **à jour** via Windows Update.
* Passer Windows en **Mode développeur** :

  1. Paramètres Windows → Mise à jour et Sécurité → Pour les développeurs → **Mode développeur**.

### **2. Activer le Sous-système Windows pour Linux**

* Panneau de configuration → Programmes et fonctionnalités → Activer ou désactiver des fonctionnalités Windows → cocher **Sous-système Windows pour Linux (bêta)**.

### **3. Finalisation**

* Redémarrer l’ordinateur.
* Lancer Bash via la barre de recherche **`bash`**.

---

## **IV) Essentiel**

* Linux : Bash est installé par défaut.
* Windows : Bash nécessite une installation via le **mode développeur** et l’activation du **Sous-système Windows pour Linux**.
* Une fois installé et lancé, Bash permet d’exécuter des commandes et scripts comme sur Linux.
