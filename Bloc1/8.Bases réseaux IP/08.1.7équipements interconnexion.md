# **Les équipements d’interconnexion**

## **Objectif**

Comprendre les différents équipements d’interconnexion (commutateurs, routeurs, points d’accès) et leurs fonctionnalités afin de concevoir, configurer et maintenir un réseau performant et sécurisé.

---

## **Concepts clés**

### **1. Définition**

* **Équipement d’interconnexion** : dispositif permettant de connecter différents éléments d’un réseau pour gérer le trafic de données, voix ou autres informations entre les nœuds du réseau.

---

### **2. Les commutateurs (Switchs)**

**Définition** : Connectent et acheminent le trafic de données entre appareils d’un réseau local (LAN) en utilisant les adresses MAC.
**Fonctionnement** :

* Table de commutation pour les adresses MAC.
* Acheminement sélectif vers le port approprié.

**Fonctionnalités** :

* **Ports console** : configuration et gestion.
* **Ports Ethernet** : connectent ordinateurs, imprimantes, serveurs.
* **Ports 10 GbE** : haut débit pour centres de données.
* **Ports SFP / SFP+** : modules optiques pour fibre, vitesses 10–25 Gb/s.
* **PoE / PoE+** : alimentation via câble Ethernet (jusqu’à 15 W / 30 W).

**Protocoles associés** :

* **Spanning Tree Protocol (STP)** : évite les boucles réseau, élit un commutateur racine, désactive les chemins redondants.
* **VLAN (Virtual LAN)** : segmentation logique d’un réseau local pour isoler le trafic.

  * Configuration : `vlan <numéro>` → `name <nom>` → `switchport access vlan <numéro>`
  * **Ports trunk** : transportent plusieurs VLAN sur un lien physique via protocole 802.1Q.

---

### **3. Les routeurs**

**Rôle** : Acheminent les paquets entre différents réseaux en choisissant le chemin optimal à partir des adresses IP.

**Types de routage** :

1. **Statique** : configuration manuelle des routes avec `ip route <réseau> <masque> <next-hop>`.
2. **Dynamique** : échanges automatiques d’informations de routage via protocoles (RIP, OSPF, BGP, IS-IS).

   * **RIP** : vecteur de distance, basé sur le nombre de sauts, limite 15 sauts.
   * **OSPF** : état de lien, segmentation en zones, algorithme Dijkstra, convergence rapide.

**Passerelle par défaut** : interface du routeur par laquelle un PC sort du réseau (`default gateway`).

---

### **4. Points d’accès Wi-Fi**

**Définition** : Permettent aux appareils sans fil de se connecter à un réseau filaire existant.

**Fonctionnement** :

1. Émission du signal radio (Wi-Fi 802.11).
2. Diffusion d’un SSID (nom du réseau).
3. Authentification via clé de sécurité (WPA2, WPA3).
4. Attribution d’une adresse IP via DHCP.
5. Transfert des données entre appareil et réseau filaire.

**Sécurité** :

* **WEP** : ancien, faible sécurité.
* **WPA / WPA2 / WPA3** : chiffrement robuste (TKIP, AES).
* Configuration sur Cisco Packet Tracer : définir SSID + clé Wi-Fi, puis connecter les appareils.

**Portails captifs** : page web obligatoire pour authentification ou acceptation des conditions avant accès réseau.

---

## **Commandes / Notes importantes**

* **Switch**

```bash
enable
configure terminal
vlan 10
name Secrétariat
interface fa0/1
switchport access vlan 10
exit
```

* **Routeur (statique)**

```bash
enable
configure terminal
ip route 192.168.1.0 255.255.255.0 20.0.0.2
```

* **Routeur (dynamique, RIP)**

```bash
enable
configure terminal
router rip
version 2
network 10.0.0.0
network 20.0.0.0
exit
```

* **Vérification** :

```bash
ping <IP>
show ip route
```

* **VLAN trunk** : utilisation de 802.1Q pour le transport multi-VLAN.

* **Wi-Fi sur AP** : configurer SSID + mot de passe (WPA2-PSK).

## **Résumé / L’essentiel**

* **Switch** : segmentation et commutation locale, PoE, STP, VLAN.
* **Routeur** : acheminement entre réseaux, statique ou dynamique (RIP, OSPF).
* **AP Wi-Fi** : accès sans fil sécurisé, DHCP, SSID, WPA2/WPA3.
* **Portail captif** : contrôle d’accès réseau.
