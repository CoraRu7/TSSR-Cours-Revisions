# Configuration et sécurisation d’un réseau sans fil (Wi-Fi)

## 1. Contexte

* Le Wi-Fi permet aux appareils (PC, smartphones, objets connectés) de se connecter sans fil à un réseau local (WLAN) et à Internet.
* **Point d’accès (AP)** : transmet/amplifie le signal Wi-Fi pour permettre la mobilité des appareils.
* Le Wi-Fi fonctionne grâce à des **fréquences radio** : 2,4 GHz et 5 GHz.

---

## 2. Principes du Wi-Fi

* Les fréquences Wi-Fi sont dans la gamme des **gigahertz (GHz)**.
* Plus la fréquence est élevée, plus la vitesse de transmission des données est importante.
* Nécessite **émetteur** (routeur/point d’accès) et **récepteur** (carte réseau Wi-Fi).

### Modes de connexion

| Mode           | Description                        | Usage typique                     |
| -------------- | ---------------------------------- | --------------------------------- |
| Infrastructure | Connexion via un point d’accès     | Particuliers, entreprises         |
| Ad-Hoc         | Appareils communiquent directement | Partage rapide entre PC ou mobile |

---

## 3. Connexion au réseau sans fil

1. Paramètres → Réseau et Internet → Wi-Fi
2. Afficher les réseaux disponibles
3. Sélectionner le réseau souhaité
4. Entrer le mot de passe du routeur/box
5. Connexion établie

---

## 4. Configuration du réseau via le routeur/box

* Accès via navigateur : `192.168.1.1` (exemple SFR)
* Authentification : identifiant + mot de passe ou bouton **WPS**
* Paramètres configurables :

  * **SSID** : nom du réseau
  * **Activation/désactivation du Wi-Fi**
  * **Diffusion du SSID**
  * **Canal et mode radio** : 2,4 GHz / 5 GHz
  * **Réseau invité**

---

## 5. Sécurisation du réseau sans fil

### Étapes clés

1. **Changer le mot de passe d’administration du routeur**

   * Empêche l’accès non autorisé à la box
   * Paramètres : Maintenance → Administration

2. **Modifier la clé Wi-Fi (mot de passe SSID)**

   * Paramètres : Wi-Fi → Sécurité

3. **Chiffrement du réseau**

   * Méthodes : WEP (faible), WPA (intermédiaire), **WPA2 (AES) recommandé**
   * Paramètres : Wi-Fi → Sécurité

4. **Changer le SSID**

   * Nom unique et non identifiable personnellement
   * Paramètres : Wi-Fi → Configuration → SSID

5. **Filtrer les adresses MAC**

   * Seuls les appareils enregistrés peuvent se connecter
   * Paramètres : Wi-Fi → Filtrage MAC

6. **Réduire la portée du signal**

   * Mode radio : ajuster selon la taille du local (ex. 802.11g pour petite surface)

7. **Mettre à jour le firmware**

   * Vérifier régulièrement sur le site du fabricant pour sécurité et performances

---

## 6. Connexion automatique au réseau sécurisé

* Une fois les paramètres configurés :

  * Entrer SSID et mot de passe sur les appareils
  * Activer l’option **Se connecter automatiquement** pour éviter de ressaisir les identifiants

---

## 7. Résumé des bonnes pratiques

* **Mot de passe complexe** pour Wi-Fi et routeur
* **Chiffrement WPA2/AES**
* **Filtrage des adresses MAC**
* **SSID non identifiable**
* **Limiter la portée si nécessaire**
* **Mises à jour régulières du firmware**
