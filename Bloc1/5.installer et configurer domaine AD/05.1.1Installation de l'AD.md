# Fiche 1 – Concepts clés : Active Directory
</summary>

## Objectif
Comprendre ce qu’est Active Directory (AD) et son rôle dans l’organisation des utilisateurs et ressources réseau.

## Définitions
- **Active Directory (AD)** : service de domaine qui centralise la gestion des utilisateurs, groupes, ordinateurs et ressources réseau.  
- **Répertoire / Base de données AD** : contient toutes les informations sur les utilisateurs, groupes, ordinateurs et ressources.  
- **Service AD DS (Active Directory Domain Services)** : rôle Windows Server permettant de gérer un domaine, les comptes et la sécurité.

## Astuces
- AD permet de déployer des politiques, sécuriser les ressources et faciliter la gestion centralisée des utilisateurs et ordinateurs.
- L’installation doit être faite sur un serveur avec une adresse IP fixe pour éviter les problèmes DNS.

# Fiche 2 – Préparation à l’installation
</summary>

## Objectif
Savoir préparer le serveur pour l’installation d’Active Directory.

## Étapes
1. Ouvrir **Gestionnaire de serveur** → **Tableau de bord**.  
2. Cliquer sur **Gérer → Ajouter des rôles et fonctionnalités**.  
3. Lire les informations préalables à l’installation → **Suivant**.  
4. Sélectionner le **type d’installation** : laisser par défaut « Installation basée sur un rôle ou une fonctionnalité » → **Suivant**.  
5. Choisir le **serveur cible** → **Suivant**.

# Fiche 3 – Installation du rôle AD DS
</summary>

## Objectif
Installer le rôle Active Directory Domain Services.

## Étapes
1. Dans la sélection des rôles, cocher **Service AD DS** → cliquer sur **Ajouter des fonctionnalités**.  
2. Vérifier que AD DS est bien coché.  
3. Cliquer sur **Suivant** pour passer à la sélection des fonctionnalités (laisser les options par défaut).  
4. Cliquer sur **Suivant** jusqu’à la page de confirmation.  
5. Cliquer sur **Installer**.  
6. Une fois terminé, cliquer sur **Fermer**.

## Astuces
- AD DS ne peut pas s’installer si certaines fonctionnalités requises ne sont pas installées → **Ajouter des fonctionnalités**.  
- L’ordinateur peut redémarrer après l’installation.

# Fiche 4 – Accès aux outils Active Directory
</summary>

## Objectif
Identifier et accéder aux outils pour administrer Active Directory.

## Étapes
1. Ouvrir **Gestionnaire de serveur** → **Tableau de bord** → **Outils**.  
2. Les outils disponibles permettent :  
   - Gestion des utilisateurs et groupes  
   - Gestion des unités organisationnelles (UO)  
   - Gestion des stratégies de groupe (GPO)  
   - Gestion du DNS intégré à AD  
   - Autres fonctionnalités administratives

## Astuces
- Après l’installation, vérifier que tous les outils AD sont présents dans **Outils** du Gestionnaire de serveur.  
- Ces outils seront utilisés pour créer les utilisateurs, groupes, UO et gérer la sécurité du domaine.
